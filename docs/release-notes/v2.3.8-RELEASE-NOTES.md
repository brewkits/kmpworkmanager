# Release v2.3.8 - Bug Fixes & Stability

## 🌟 Highlights
- **iOS Stability:** Fixed a critical concurrency bug in `ChainExecutor` where old tasks wouldn't cancel correctly when applying a `REPLACE` policy.
- **Security Updates:** Tightened `SecurityValidator`'s check for malformed IPv6 loopback addresses to prevent potential SSRF bypasses.
- **iOS Test Infrastructure:** Refactored various suspend functions in tests to use `runTest`, addressing test instability. Extracted `IosFileStorage` instantiation to support better isolated testing for iOS native components.
- **Queue Stability:** Fixed `ArrayIndexOutOfBoundsException` occurring during enqueue operations with empty string/byte arrays.

## 🐛 Bug Fixes
- **iOS REPLACE Policy:** Migrated task cancellation logic into the `NativeTaskScheduler` to reliably terminate running jobs during chain replacement.
- **SecurityValidator IPv6 Checks:** Improved IPv6 validation against `::` and edge cases where malformed formats like `[::1::1]` were previously slipping through.
- **AppendOnlyQueue (iOS):** Fixed a crash issue in `toNSData()` when serializing empty byte arrays. Enabled the storage engine to handle zero-length payloads properly.

## 🛠 Refactoring & Cleanup
- Removed temporary and debug comments from the test suite.
- Cleaned up obsolete Vietnamese annotations.
- Updated documentation versions to 2.3.8.
- Made iOS internal classes `IosFileStorage`, `ChainTransaction`, and `IosFileStorageConfig` public to support testing.
