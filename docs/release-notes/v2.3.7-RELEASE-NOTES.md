# Release v2.3.7 - Stability and Documentation Updates

**Release Date:** March 24, 2026

## 🚀 Highlights

This release focuses on addressing issues found during the review process, improving documentation accuracy, and ensuring test stability.

## ✨ New Features & Improvements

### Core
- **Test Stability:** Addressed test failures in `kmpworker` related to `TaskEventBus` replay configuration.
- **KSP Test Fixes:** Implemented mock classes for KSP tests to bypass Kotlin version incompatibilities, though tests are still marked as failing due to environment issues.

## 🐛 Bug Fixes

### Critical Fixes
- **`TaskEventBus` Replay:** Corrected `TaskEventBus` test to match the intended `replay = 1` behavior.

### Documentation Updates
- **`api-reference.md`:** Updated to reflect `JsonObject` usage for `WorkerResult.Success.data` and `TaskCompletionEvent.outputData`. Also corrected `TaskEventBus.emit` signature and examples.
- **`CHANGELOG.md`:** Corrected release date for v2.3.7.

## 📊 Test Results

### `kmpworker` Module:
- All tests passing after fixes to `TaskEventBusTest`.

### `kmpworker-ksp` Module:
- **Known Issue:** All tests are currently failing due to Kotlin binary compatibility issues between `kotlin-compile-testing` and the project's Kotlin version. This is a known environment limitation and will be addressed in a future update. Mocks have been implemented to isolate the KSP processor logic, but the test runner fails due to environment incompatibility.

## 🔄 Migration Guide

No breaking changes in this release. Safe to upgrade.
