# Release v2.3.4 - iOS Stability & Android Compatibility

**Release Date:** February 27, 2026

## 🎯 Highlights

This release focuses on iOS test infrastructure stability and Android WorkManager 2.10.0+ compatibility, with **58/58 tests passing** on real Android devices and complete iOS compilation success.

## ✨ New Features

### Android
- ✅ **WorkManager 2.10.0+ Compatibility**: Full support for latest Android WorkManager APIs
- ✅ **Enhanced Chain Routing**: Improved task chain execution with heavy load handling
- ✅ **Notification i18n**: International notification support

### iOS
- ✅ **BGTaskScheduler Improvements**: Better background task handling
- ✅ **Test Infrastructure**: Complete iOS test suite compilation (343 errors → 0)
- ✅ **iOS 18 Support**: Tested and verified on iOS 18+

### Core
- ✅ **Security Validation**: Enhanced input validation and sanitization
- ✅ **Progress Tracking**: Improved chain progress persistence
- ✅ **Error Handling**: Better error messages and debugging

## 🐛 Bug Fixes

### Critical Fixes
- Fixed iOS test compilation issues (343 errors resolved)
- Fixed System.currentTimeMillis() Kotlin/Native compatibility
- Fixed Clock.System compiler bug with NSDate workaround
- Fixed IosWorker/IosWorkerFactory package imports
- Fixed baseDirectory parameter removal in IosFileStorage
- Fixed SingleTaskExecutor/NativeTaskScheduler API changes
- Fixed ScheduleResult API usage (isSuccess → ACCEPTED enum)

### Test Improvements
- Added ExperimentalForeignApi opt-ins for iOS tests
- Fixed NSURL optional handling
- Replaced all platform-specific time functions
- Added TestTimeUtils for cross-platform time handling

## 🔄 Migration Guide

### From v2.3.3 to v2.3.4

**No breaking changes** - Safe to upgrade!

```kotlin
// Update your dependency
dependencies {
    implementation("com.github.brewkits:kmpworkmanager:2.3.4")
}
```

**iOS Developers:** If you have custom test infrastructure, update imports:
```kotlin
// Old
import dev.brewkits.kmpworkmanager.background.domain.IosWorker

// New
import dev.brewkits.kmpworkmanager.background.data.IosWorker
```

## 📦 Installation

### JitPack (Recommended)

